CA-SDK CLIENT

INTRODUCTION

CA-SDK Client is an app which allows developers to evaluate the features and sensor capabilities of the InvenSense CA-SDK platform. The InvenSense CA-SDK solution delivers a sensor platform capable of tracking attitude by combining the InvenSense MPU-9250 9-axis MotionTracking device (gyroscope/accelerometer/magnetometer) with an onboard microcontroller and Bluetooth radio module, in addition, it includes external sensors of Light, Humidity, Temperature, Pressure, Proximity, and UV on a single board. This app can be made available in source code for developers of custom specific code for CA-SDK developers. For further information, please register at http://www.invensense.com/developers. To purchase the CA-SDK platform, please visit http://www.invensense.com/mems/gyro/nineaxis.html

SENSORS tab is intended to display the status and values of different sensors while DISPLAY tab is intended to illustrate the capabilities of eMPL.

COMMANDS DESCRIPTION

  1. Accelerometer Sensor: Turn on/off the accelerometer sensor only when DMP is not ON.

  2. Gyro Sensor: Turn on/off the gyroscope sensor only when DMP is not ON.

  3. Compass Sensor: Turn on/off the compass sensor only when DMP is not ON.

  4. Accelerometer Values Display: Toggle the status of printing acceleration in the display list section.

  5. Gyro Values Display: Toggle the status of printing gyro values in the display list section.

  6. Compass Values Display: Toggle the status of printing compass values in the display list section.

  7. Euler Values Display: Toggle the status of printing euler values in the display list section.

  8. Rot. Matrix Values Display: Toggle the status of printing rotation matrix values in the display list section.

  9. Quaternion Values Display: Toggle the status of printing quaternion values in the display list section.

  10. Heading Value Display: Toggle the status of printing heading value in the display list section.

  11. Pedometer Display: Toggle the status of printing walk count values in the messages list section, also reset the walk count and pedometer display to zero.

  12. Other Sensors Display: Toggle the status of printing pressure, temperature, humidity, UV Index values in the display list section.

  13. Magnetic Disturbance: Turn on/off the magnetic disturbance for CA-SDK.

  14. Low Power Quaternion: Toggle low power quaternions.

  15. DMP: Turn on/off DMP.

  16. Dump Gyro Register Values: Read the gyro register value and will show it in the messages list section.

  17. Test Low Power Accel Values: Test low power accel values and will display the result in messages list section.

  18. Load Calib Data: Load the calibration values of gyro from internal flash memory.

  19. Store Calib Data: Store the calibration values of gyro into internal flash memory.

  20. Self Test: Run the self test and display the result in messages list section

  21. Set Sensor Data Rate to 10Hz: If DMP is on then it will set the DMP FIFO rate to 10 Hz else it will set the gyro sample rate to 10Hz.

  22. Set Sensor Data Rate to 20Hz: Same as for command no. 21 except with frequency of 20 Hz.

  23. Set Sensor Data Rate to 40Hz: Same as for command no. 22 except with frequency of 40 Hz.

  24. Set Sensor Data Rate to 50Hz: Same as for command no. 21 except with frequency of 50 Hz.

  25. Set Interrupt on Gesture Event Only: Set hardware to interrupt on gesture event only. This feature is useful for keeping the MCU asleep until the DMP detects as a tap or orientation event.

  26. Set Interrupt Periodically: Set hardware to interrupt periodically

  27. Reset Pedometer: Reset the pedometer count and walk time to zero.

  28. Reset Board: Reset the whole CA-SDK hardware and initialize all the parameters from the start.

  29. Set Motion Interrupt Mode: Set the hardware to interrupt on motion and no-motion event.

  30. Get eMPL Version: Fetch the eMPL version and display it in the messages list section.

  31. Get Compass Status: Fetch the compass accuracy and status and display it in the messages list section.
